# MeetingGhost Gold — Roadmap

## ✅ Completed (v1.0 - v4.0)
- **v1.0 (Init):** Setup Vite + React + Capacitor project.
- **v2.0 (Native Polish):** UI/UX foundations, brushed gold aesthetic, audio waveform visualization.
- **v3.0 (On-Device AI Workers):** Web Worker architecture for Whisper (STT) and TinyLlama (Summarization).
- **v4.0 (UX & Pro Audio):** iOS `100dvh` fixes, Floating Action Area, First-Time Onboarding UI, Audio File Import, Auto-Titling, PDF/Markdown Exports, and History Search.

## 🚀 Next Up: v5.0 (Organization & Intelligence)
*See `TASKS.md` for specific tickets.*

- Translation capabilities (Spanish/French).
- Folder organization for the History tab.
- JSON Cloud Sync (Import/Export `localStorage`).
- Advanced Audio Visualizer Themes.

## 🧠 Future Visions: v6.0 and Beyond
- **P2P Encrypted Local Sync:** Sync between PWA and mobile devices on the same WiFi using WebRTC.
- **Enterprise Integrations:** Push action items to Jira/GitHub natively.
- **Capacitor Native ML Core:** Shift from WebAssembly to Apple CoreML and Android NNAPI for huge speed boosts on mobile native apps.
